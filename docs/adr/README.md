## ADRフォーマット

各ADRは以下の構造に従っています：

- **タイトル**: 決定の簡潔な説明
- **ステータス**: 提案中/承認済み/廃止/置き換え
- **コンテキスト**: 決定が必要な背景
- **検討した選択肢**: 評価した代替案
- **決定**: 選択した解決策
- **理由**: なぜその決定を行ったか
- **影響**: 決定の結果として生じる変更

## ステータス

- **提案中**: レビュー待ち
- **承認済み**: チームで合意済み
- **廃止**: 無効化された決定
- **置き換え**: 他のADRで更新された決定
